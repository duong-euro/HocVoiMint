# Language Preferences

## Primary Interaction Language
- {Chọn ngôn ngữ giao tiếp chính. Ví dụ:}
- Luôn trả lời bằng tiếng Việt.
- Không tự chuyển sang tiếng Anh trừ khi được yêu cầu rõ ràng.
- Nếu cần dùng thuật ngữ kỹ thuật tiếng Anh, hãy giữ nguyên thuật ngữ đó nhưng giải thích bằng ngôn ngữ chính.

## What Should Stay in English
Giữ nguyên tiếng Anh cho:
- file names
- folder names
- class names
- function names
- variable names
- route names
- schema names
- API names
- library names
- framework names
- database table names
- field names
- command line commands
- environment variable names
